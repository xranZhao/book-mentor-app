# Book Mentor · 书本智慧顾问

一个 PWA 手机应用，将 13 本心理学/自我成长/亲密关系书籍的应用框架装进手机。

## 功能

- 📚 浏览 13 本书的书库（按标签筛选、关键词搜索）
- 💬 选一本书，用它的视角分析你的生活困惑
- ⚙️ 配置你的 DeepSeek API Key

## 本地运行

```bash
# 方式1：双击 start-server.bat
# 方式2：命令行
python -m http.server 8080
```

然后浏览器打开 `http://localhost:8080`

## 手机安装

1. 部署到 GitHub Pages 后，用手机浏览器打开 URL
2. Safari (iOS) 点击「分享」→「添加到主屏幕」
3. Chrome (Android) 点击「⋮」→「添加到主屏幕」
4. 之后可以像原生 App 一样使用

## 技术架构

- 纯前端 PWA（无需后端服务器）
- Vanilla JS + CSS（零框架依赖）
- Service Worker 离线缓存
- DeepSeek API 驱动 AI 对话（framework 注入 system prompt）
